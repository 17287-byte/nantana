ผลงานและกิจกรรม

รวบรวมผลงาน กิจกรรม และประสบการณ์ที่ภาคภูมิใจระหว่างการศึกษา

🏆 ผลงานที่ภาคภูมิใจ

  • เข้าร่วมการแข่งขันทักษะวิชาการ
  • ได้รับเกียรติบัตรจากการอบรมและกิจกรรม
  • สร้างเว็บไซต์ Portfolio ด้วย HTML และ CSS
  • ฝึกใช้งาน GitHub สำหรับเผยแพร่เว็บไซต์

🌟 กิจกรรมที่เข้าร่วม

  • กิจกรรมจิตอาสา
  • กิจกรรมวันสำคัญของโรงเรียน
  • ค่ายวิชาการ
  • กิจกรรมพัฒนาทักษะความเป็นผู้นำ

💡 ทักษะที่ได้รับ

  • การทำงานเป็นทีม
  • ความรับผิดชอบ
  • การแก้ไขปัญหา
  • HTML และ CSS เบื้องต้น
  • การใช้ Git และ GitHub

📸 ตัวอย่างผลงาน

เกียรติบัตรที่1

เป็นการแข่งขันประกวดวาดภาพระบายสีตามจินตนาการเนื่องในกิจกรรมวันวิทยาศาสตร์

เกียรติบัตรที่2

รูปภาพหรือเกียรติบัตรจากการแข่งขัน หรือการเข้าร่วมอบรม

เกียรติบัตรที่3

ภาพกิจกรรมจิตอาสา กิจกรรมวิชาการ หรือกิจกรรมภายในโรงเรียน

เกียรติบัตรที่4

สามารถเพิ่มรูปผลงาน โครงงาน หรือกิจกรรมที่ภาคภูมิใจได้
